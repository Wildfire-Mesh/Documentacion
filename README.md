# WildFire Mesh - Documentación Central

Somos una organización que busca brindar una alternativa de comunicación confiable en entornos donde las redes tradicionales suelen fallar, como zonas de cerros y montañas. Nuestro objetivo es que combatientes del fuego y equipos de rescate cuenten con un canal adicional de comunicación crítica para coordinarse y actuar de manera más segura y eficiente durante emergencias forestales.

## 🎯 Objetivo

Centralizar el conocimiento del proyecto en:

- Experimentos de comunicación.  
- Criterio de instalación de nodos.  
- Configuraciones recomendadas.  
- Lecciones aprendidas.  
- Guías operativas.  
- Traducciones al castellano.

## 🗂️ Estructura de la documentación

- Blogs: Relatos de nuestro proceso de aprendizaje, experiencias de campo, y encuentros con la comunidad.  
- Docs: Guías de contenido.

## 🛠️ Tecnologías y herramientas

- **Firmware:** Integración con el ecosistema [*MeshCore*](https://meshcore.co.uk) y [*Meshtastic*](https://meshtastic.org).  
- **Hardware:** Nodos con chips nRF25 y ESP32 y módulos de radio LoRa.  
- [**Docusaurus:**](https://docusaurus.io) Generador de sitio estático.  
- **Markdown:** Fuente de documentación.  
- **Node.js:** Entornos de comunicación.
